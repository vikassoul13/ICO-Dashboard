TokenWiz - ICO User Dashboard Admin Template

  • Version: 1.0.4
  • Updated: 03/13/2019
  • By: Softnio

A powerful admin template for ICO Dashboard with all kind of components, features, pre-build pages such as Purchase Tokens, Profile, KYC Form, KYC Lists, User Lists, Transaction Lists and many more.

TokenWiz is based on Bootstrap 4, So it give you more flexible for customize. All files are clearly organised we believe it will be easy to use and edit them. Any one can customize this template as there wishes.

If you have any questions that are beyond the scope of this help file, please feel free to email via our user profile page contact form here. Thanks so much!

Geting Started

In our package folder you will see a file called _blank.html, Just duplicate this page and start adding your content.

Please check our elements page for styled html elements and pre-build components. Also all shorts of pages designed specily for ico management system, you can also start with pre-build pages and edit them as you see fit.

Before editing any file please keep a copy of that file for referance.

General Elements

Use .input-bordered class with input element to make it bordered.

Use .input-textarea as an extra class with your desired input class for textarea.
Also you can use .input-textarea-sm as additional class for smaller height.

Default Select Box .input-bordered for styling

Use .select class for initializing select2 jquery plugins. And .select-bordered or .select-solid for styling

Use .input-file class with input[type="file] for file upload form.
Also you can use .input-file-sm for smaller sized file.

Use .date-picker class for initializing date picker jquery plugin.

Use .time-picker class for initializing date picker jquery plugin.

Other Elements

Use .input-checkbox class with input[type="checkbox] for custom styled checkbox.
Also you can use .input-checkbox-sm for smaller sized checkbox.

Use .input-switch class with input[type="checkbox] for switch style checkbox.
Also you can use .input-switch-sm for smaller sized switch.

Use .input-radio class with input[type="radio] for custom styled Radio.
Also you can use .input-radio-sm for smaller sized Radio.

Use .input-switch class with input[type="radio] for switch styled Radio.
Also you can use .input-switch-sm for smaller sized switch.
